This depository includes data and a MATLAB function for adjusting paleointensity site scatter for the number of specimens used to obtain the average result.
This follow the work of Paterson et al. (2010)

If any problems or questions arise please e-mail me (greig.paterson@mail.iggcas.ac.cn)


References:
Paterson, G. A., D. Heslop, and A. R. Muxworthy (2010), Deriving confidence in paleointensity estimates, Geochem. Geophys. Geosyst., 11, Q07Z18, doi:10.1029/2010GC003071.
